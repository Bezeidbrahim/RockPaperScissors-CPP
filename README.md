# ✂️ Rock Paper Scissors — C++

> A console-based Rock Paper Scissors game where you play against the computer over multiple rounds.

---

## 🎮 How It Works

- You choose **Stone**, **Paper**, or **Scissors**
- The computer picks randomly
- Best of 3 rounds wins the game
- You can replay as many times as you want
